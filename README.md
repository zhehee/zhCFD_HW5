# zhCFD_HW5
计算流体力学第五次课程作业

## 前置要求

- Python 3.7+
- matplotlib,numpy第三方库
- texlive/simple Tex

## 主要内容

- 关于第五次作业的报告
- 关于第五次作业的代码





## 基本使用：

- 在hw5.tex查看报告的 latex 源码
- 查看由latex编译的hw5.pdf报告

- 在commit.jpg查看git版本控制记录，也可以直接访问github网址https://github.com/zhehee/zhCFD_HW5
